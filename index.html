A/B Testing

Overview

We were given a copy of a website to modify and test to see if our new version, Version B, was better than the original, Version A.

In order to assess the effectiveness of the websites, users had to complete a task, which was to schedule an appointment with Adam Ng, MD at Morristown Medical Center on April 23, 2024.

Creating Hypotheses

‚ùå Misclick rate

  • Null: There will be no difference between the frequency of misclicks on Version A versus Version B.
  • Alternative: There will be a difference between the frequency of misclicks on Version A versus Version B.
  • Prediction: I believe that the null hypothesis will not be rejected because I think it's likely for a user to misclick when they are under a time constraint, but in this test, there is none. Therefore, I believe that it will simply take a user longer to find the correct "schedule appointment" button without compromising accuracy.

üïí Time on page

  • Null: The amount of time a user spends to complete the task on Version A will be the same as the time a user spends on Version B.
  • Alternative: The amount of time a user spends to complete the task on Version A will be different than the time a user spends on Version B.
  • Prediction: I believe that the null hypothesis will be rejected because the increased usability in Version B will guide the user and ultimately make it easier for them to complete the task. The clear distinction between each row using a background color, bolder dates, and buttons with more contrast will hopefully decrease the time it takes for a user to complete the task on Version B compared for Version A.

üñ±Ô∏è Total mouse distance

  • Null: There will be no difference between the total distance traveled in pixels by the user's mouse on Version A versus Version B.
  • Alternative: There will be a difference between the total distance traveled in pixels by the user's mouse on Version A versus Version B.
  • Prediction: I believe that the null hypothesis will be rejected because due to the increased usability stated previously, the user will spend less time trying to navigate Version B with their mouse. Additionally, since users are testing Version B after they have already gained familiarity with and tested Version A, their experience will enable them to find the correct appointment faster.

I chose the chi-squared test to compare two categorical variables, the number of misclicks versus non-misclicks across trials in Version A and Version B. This test was most applicable in testing whether or not there is a statistically significant difference across the two versions. Based on the output, this is indeed true because I got a p-value of 0.005, which is less than the standard alpha value of 0.05. Therefore, there is a 0.5% chance that there is no significant difference between the rate of misclicks across Versions A and B, which means that we reject the null hypothesis and conclude that there is indeed a significant difference between Versions A and B.

I chose the two-tailed t-test to determine whether or not there was a difference between Versions A and B in terms of the amount of time the user spent on the page because I wanted to know if there was a statistically significant difference to then conduct a one-tailed t-test. After computing the results, I ended up with a p-value of 0.18, which is less than the alpha value of 0.05. Therefore, we fail to reject the null hypothesis and there is not enough evidence to conclude that there is a difference between Version A and B in terms of the time it takes for a user to complete the task.

Summary Statistics

In total, I collected 30 data points for Version B while we collected 34 for Version A. Based on my computations, I concluded that there we can reject the null hypotheses for the misclick rate and total mouse distance; however, we failed to reject the null hypothesis for time on page.

  • The ratio between users who misclicked versus those who did not is much higher in Version A than Version B, indicating that Version B is less error-prone.
  • In Version A, the total mouse distance a user travels on average is about 1.42 times more pixels than Version B. Version B has a smaller variance than A, which indicates that the data is closer to this average in Version A.

Based on both the misclick rate and total mouse distance, we can conclude that Version B is likely more effective than Version A.
